# Social-Sentry-PC-

A Windows native application for social media monitoring and security.

## Description

Social Sentry PC is a desktop application built for Windows that provides comprehensive social media monitoring and security features.

## Technologies

- Windows Presentation Foundation (WPF) / WinUI
- .NET Framework / .NET Core
- C#

## Getting Started

### Prerequisites

- Visual Studio 2022 or later
- .NET SDK
- Windows 10/11

### Installation

1. Clone the repository
2. Open `Social Sentry.sln` in Visual Studio
3. Build and run the solution
